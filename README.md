# Sistema de Monitoramento Residencial Inteligente

## Descrição do Projeto
Este projeto integra hardware (Arduino) e software (Java) para monitorar temperatura, umidade e luminosidade residencial. Os dados são capturados por sensores, processados por uma API Java e visualizados em um protótipo mobile.

## Tecnologias Utilizadas
* **Hardware:** Arduino Uno, Sensores TMP36 e LDR.
* **Linguagem:** Java.
* **IDE:** NetBeans.
* **Prototipagem:** Figma e Tinkercad.

## Estrutura de Arquivos
* `/src`: Código-fonte da aplicação Java.
* `/docs`: Diagramas UML e documentação extra.
